## 目的
导航页的后台，可以管理用户，用户生成的accesskey，用户的备份数据

## 要求
- 使用go语言编写，开发go-web程序，使用sqlite存储数据；
- 程序启动使用时--user --pwd 指定最高权限用户的凭证信息，两个参数必须同时存在。如果没有这两个参数就自动生成master用户密码在日志里面，用户第一次登录就修改密码，不设置密码策略，写入数据库密码不做加密；
- port参数指定程序端口，默认8445
- 角色只区分普通用户和管理员

## 后台页面菜单
+ 用户管理：可以添加删除所有的用户（删除用户的时候，必须查看用户是否存在备份数据或者密钥，如果有需要先删除备份数据和密钥）
+ 密钥管理：查看所有或者个人的密钥（根据权限），创建，删除，密钥过期
    - 查看字段：access key,secret key 创建时间，过期时间，创建人
    - 创建密钥：可以设置密钥过期的天数，默认30天，选项1，7，15，30，180，360，永久
    - 密钥过期：立马让其密钥过期
+ 备份管理：查看所有人备份，创建，删除，详情，下载
    - 查看字段：备份名称（唯一值）、上传时间，更新时间，备份人，备份大小，同步次数
    - 下载此记录json数据
+ 同步记录：查看所有的同步记录，清理记录
    - 字段：备份名称，传输类型(上传下载)，所使用key，操作人
    - 清理记录：选择清理天数，3，7，30，全部
## 远程同步接口
注意：同步接口是使用header里面x-access-key，x-secret-key进行认证的
- /api/sync/list 返回当前用户的所有数据列表
- /api/sync/download/{id} 返回这个备份json数据
- /api/sync/upload 用户上传备份数据json格式

## 导航页同步的信息-数据结构详解
### Partition（工作区/分区）
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | number | 是 | 分区唯一ID |
| name | string | 是 | 分区名称 |
| order | number | 否 | 排序顺序，默认0 |
| isPrivate | boolean | 否 | 是否为隐私分区，默认false |

### Folder（文件夹）
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | number | 是 | 文件夹唯一ID |
| name | string | 是 | 文件夹名称 |
| collapsed | boolean | 否 | 是否折叠，默认false |
| partitionId | number | 否 | 所属分区ID |
| isPrivate | boolean | 否 | 是否为隐私文件夹 |

### Shortcut（书签）
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | number | 是 | 书签唯一ID |
| name | string | 是 | 书签名称 |
| url | string | 是 | 书签URL |
| icon | string | 否 | 自定义图标URL，空则使用favicon |
| folderId | number | 否 | 所属文件夹ID，null表示根目录 |
| partitionId | number | 否 | 所属分区ID |
| isPrivate | boolean | 否 | 是否为隐私书签 |
| isPinned | boolean | 否 | 是否置顶 |

### SearchEngine（搜索引擎）
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | number | 是 | 搜索引擎唯一ID |
| name | string | 是 | 搜索引擎名称 |
| url | string | 是 | 搜索URL模板，`%s` 为搜索词占位符 |
| icon | string | 否 | 图标URL |

### Settings（外观设置）
| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| bgType | string | "gradient" | 背景类型：gradient/solid/image |
| gradientColor1 | string | "#667eea" | 渐变起始颜色 |
| gradientColor2 | string | "#764ba2" | 渐变结束颜色 |
| gradientAngle | number | 135 | 渐变角度（0-360） |
| solidColor | string | "#1a1a2e" | 纯色背景颜色 |
| bgImage | string | "" | 背景图片URL或Base64 |
| iconSize | number | 57 | 图标大小（px） |
| folderSize | number | 65 | 文件夹大小（px） |
| iconGap | number | 15 | 图标间距（px） |
| folderGap | number | 20 | 文件夹间距（px） |
| iconRadius | number | 11 | 图标圆角（px） |
| searchRadius | number | 16 | 搜索框圆角（px） |
| btnRadius | number | 12 | 按钮圆角（px） |
